Simon Game
A browser-based memory game inspired by the classic electronic game Simon. Repeat the sequence of colors and sounds to test your memory and reflexes.

🎮 How It Works
Press any key on your keyboard to start the game.
Watch and listen to the sequence of colors.
Click the buttons in the same order.
Each level adds a new color to the sequence.
If you make a mistake, the game ends and restarts.

⚠️ Note
This game is designed for desktop browsers. Mobile or touch devices may not start the game because it requires a physical keyboard.

🚀 Features
Interactive button animations
Sound effects for each color
Progressive difficulty with levels
Easy restart functionality
